# Your Name in Landsat

A CC0* re-implementation of NASA's [Your Name in Landsat](https://science.nasa.gov/specials/your-name-in-landsat/) site<br/><sub>*: Images are public domain by default as a work of NASA/USGS</sub>

Built as a hobby project, not serious. For transparency, the design and some of the code was based on https://github.com/eudoxia0/etudes/tree/master/html/boringpunk-notes-app, which is vibecoded to shit with Claude. However most of the Landsat specific stuff is either written by yours truly or re-implemented from the minified code of the original Your Name in Landsat (again, public domain by default, made by NASA). Oh, and the mobile support, I did that (and I'm very proud of it!!! I'm usually awful at mobile support!!!!)

Available on GitHub Pages at https://coleisforrobot.github.io/your-name-in-landsat/
